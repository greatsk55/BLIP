BLIP의 주요 변경 사항을 기록합니다.

## v1.7.0 — 범용 파일 전송

- **모든 파일 형식 지원**: PDF, 문서, 압축 파일, 코드 파일 등 전송 가능
- **스마트 파일 아이콘**: 확장자 기반 자동 아이콘 선택
- **파일 크기 검증**: 유형별 제한 (이미지 50MB, 동영상 100MB, 파일 200MB)
- **다운로드 버튼**: 수신 파일 원클릭 다운로드
- **전송 진행률**: 실시간 전송 진행 표시
- **종단간 암호화**: 메시지와 동일한 E2E 암호화 적용

## v1.6.0 — 그룹 채팅

- **그룹 방**: 참여자 수 제한 없는 그룹 채팅방 생성
- **관리자 기능**: 방 잠금, 강퇴, 차단, 폭파 기능
- **관리자 토큰**: 방 관리를 위한 별도 관리자 인증
- **참여자 사이드바**: 실시간 참여자 목록 및 관리자 표시
- **공유 링크**: 비밀번호 포함 선택 가능한 초대 링크 생성
- **24시간 만료**: 그룹방 24시간 후 자동 만료

## v1.5.0 — 임베드 가능한 채팅 위젯

- **임베드 가이드**: 모든 웹사이트에 BLIP 채팅을 삽입하는 단계별 가이드
- **위젯 커스터마이징**: 외관 및 동작 설정 가능
- **크로스 오리진 지원**: 적절한 CSP 헤더를 통한 안전한 iframe 삽입

## v1.4.0 — 커뮤니티 초대 코드 시스템

- **초대 코드**: 커뮤니티 초대 코드 생성 및 공유
- **커뮤니티 게시판**: 커뮤니티 토론을 위한 공개 게시판
- **속도 제한**: 스팸 및 악용 방지

## v1.3.0 — 모바일 앱 & 안정성

- **모바일 최적화**: 완전한 반응형 디자인 및 PWA 지원
- **비주얼 뷰포트 처리**: 모바일 기기에서의 올바른 키보드 동작
- **성능 개선**: 렌더링 및 메모리 사용량 최적화

## v1.2.0 — 암호화 미디어 전송

- **이미지 전송**: E2E 암호화된 이미지 송수신
- **동영상 전송**: 썸네일이 포함된 암호화 동영상 공유
- **청크 기반 전송**: 안정적 전달을 위한 대용량 파일 분할
- **체크섬 검증**: 모든 전송에 SHA-256 무결성 검증

## v1.1.0 — 즉시 방 폭파 & 알림

- **방 폭파**: 채팅 완료 후 즉시 방 폭파
- **브라우저 알림**: 메시지 도착 시 알림
- **화면 캡처 감지**: 화면 캡처 감지 시 경고
- **퇴장 확인**: 채팅방 퇴장 전 확인

## v1.0.0 — 최초 릴리스

- **종단간 암호화**: X25519 키 교환 + XSalsa20-Poly1305 암호화
- **임시 채팅방**: 메시지 미저장, 자동 만료
- **비밀번호 기반 접근**: 생성된 비밀번호로 안전한 방 접근
- **실시간 메시징**: Supabase Realtime 기반 WebSocket 즉시 메시징
- **제로 지식 아키텍처**: 서버가 평문 메시지를 절대 볼 수 없음
- **다국어 지원**: 영어, 한국어, 일본어, 중국어, 스페인어, 프랑스어
