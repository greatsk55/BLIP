BLIP은 흔적을 남기지 않는 일시적 채팅 서비스입니다.
계정도, 기록도, 프로필도 없습니다. 링크 하나로 시작하고, 끝나면 모든 것이 사라집니다.

> "이 대화는 남길 필요가 없다" — 그런 순간을 위해 만들어졌습니다.

---

## Why BLIP?

기존 메신저는 너무 많은 것을 남깁니다. 계정, 친구 목록, 대화 기록, 알림...
하지만 세상에는 **잠깐 말하고 사라지면 되는 대화**가 훨씬 많습니다.

| 기존 메신저 | BLIP |
|---|---|
| 계정 생성 필요 | 계정 없음 |
| 대화 기록 영구 저장 | 기록 zero — 복구 불가 |
| 친구 추가 필요 | 링크 하나로 즉시 참여 |
| 서버에 데이터 보관 | 서버 저장 없음, 종단간 암호화 |

## 핵심 컨셉

- **0 준비 비용** — 링크 하나로 즉시 시작
- **0 지속성** — 끝나면 복구 불가
- **0 정체성** — 계정, 친구, 프로필 없음
- **100% 합의 기반** — 참여자 모두 동의한 순간만 존재

## 사용 방법

```
1. 방 만들기  →  버튼 하나로 생성
2. 링크 공유  →  상대방에게 전달
3. 대화       →  종단간 암호화 실시간 채팅
4. 종료       →  모든 데이터 즉시 파쇄
```

## 사용 사례

- "잠깐 얘기하고 방 폭파"
- "작전 회의 후 흔적 제거"
- "링크 하나로 즉시 모임"
- 게임 파티 조율, 이벤트 현장 소통, 민감한 일회성 대화
